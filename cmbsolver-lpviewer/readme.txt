A plugin to view page information with image files and text content.

== Description ==

CMB Solver LP Viewer displays page information with four tabs:
1. Image file
2. Text with line breaks
3. Text without line breaks
4. Runeglish transposed text

Use the shortcode `[lpviewer]` to display the viewer on any page or post.

== Installation ==

1. Upload the `cmbsolver-lpviewer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the shortcode `[lpviewer]` in your pages or posts

== Usage ==

Basic usage:
`[lpviewer]`

With default page selected:
`[lpviewer default_page="5"]`
